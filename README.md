# 個人化資產配置與管理

這是一個只讀取 Google Sheet 的 Dashboard。本機伺服器會把 Google Sheet 匯出的 CSV 讀進瀏覽器端，產生資產摘要、資產配置、帳戶分布與明細表，不會修改原始 Google Sheet。

## Google Sheet 欄位建議

第一列請放欄位名稱，建議使用以下欄位：

| 欄位 | 說明 |
| --- | --- |
| 資產類別 | 例如台股、美股、現金、債券、基金 |
| 資產名稱 | 例如 0050、VOO、活存 |
| 帳戶 | 例如銀行、券商或信託帳戶 |
| 數量 | 持有單位數 |
| 單價 | 單位價格 |
| 市值 | 目前市值，Dashboard 會優先用這個欄位統計 |
| 幣別 | 例如 TWD、USD |

台股持股表也支援 `股票代號`、`股票名稱`、`持有股數`、`庫存股數`、`現價`、`收盤價`、`庫存市值`。如果沒有市值欄位，Dashboard 會用股數乘以現價估算。

也支援部分英文欄位，例如 `Asset Class`、`Asset`、`Account`、`Quantity`、`Price`、`Value`、`Currency`。

## 使用方式

1. 將 Google Sheet 分享權限設為「知道連結的任何人可檢視」。
2. 在此資料夾執行：

```bash
node server.js
```

3. 開啟：

```text
http://localhost:5173
```

4. 開啟頁面後會自動載入固定資料來源。
5. 需要更新時，按下「重新整理資料」。

Google Sheet 網址中的 `gid` 通常在網址最後方，例如：

```text
https://docs.google.com/spreadsheets/d/試算表_ID/edit#gid=0
```

## 注意事項

- 目前版本適合公開或半公開分享連結的 Google Sheet。
- Dashboard 透過本機只讀代理抓取 CSV，可避免瀏覽器 CORS 限制。
- 如果你的 Sheet 必須保持私人權限，需要再加上後端服務與 Google OAuth 或 Service Account。
- 台股會優先使用 `尚存股數` 與 `現值`，避免部分賣出後仍用原始股數計算。
- 美股會從 `外幣帳戶` 或 `外幣帳戶整理` 的股票摘要區讀取剩餘股數與目前價格，已清倉標的會自動排除。
- 美股會使用 `外幣帳戶整理` 摘要區的 USD/TWD 匯率換算成台幣，總資產、配置比例與帳戶分布都以台幣基準計算。
- 上方分頁可切換目前持股、台股已實現損益、美股已實現損益；已實現損益分頁會用同樣的摘要、圖表與明細表設計呈現。
